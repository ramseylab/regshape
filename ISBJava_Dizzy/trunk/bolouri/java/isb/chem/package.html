This package contains classes used to simulate the kinetics of chemical reactions involving one or more chemical species. These reactions have reactants and products, which are sets of chemical species. A collection of reactions together constitutes a model. Given a model and a collection of initial populations for the chemical species in the model, you may solve for the dynamics of the model using one of several algorithms. The only algorithm currently implemented in this package is the Gillespie stochastic algorithm.

SBML Level 1 Specification

The classes in this package have been designed for optimum compatibility with the SBML Level 1 specification, as described in the following publication:
M. Hucka, A. Finney, H. Sauro, and H. Bolouri. Systems Biology Markup Language (SBML) Level 1: Structures and Facilities for Basic Model Definitions. 2 March 2001.
An electronic version of this document is available at www.sbw-sbml.org.
