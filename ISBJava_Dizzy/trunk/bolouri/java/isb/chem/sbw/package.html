This package contains classes used to provide an interface to the isb.chem package via the Systems Biology Workbench (SBW) system. The overview for the SBW system is available on the web. The SBW system also provides a facility for loading a model described in an SBML document, and generating an equivalent model description in the Command Language of the isb.chem.scripting package.

In order to compile this package, you will need to have the SBW system installed on the build system and the target (runtime) system. In particular, this package makes use of the JAR files 

/usr/local/lib/SBWCore.jar
/usr/local/modules/SBWValidate.jar
whre "/usr/local" should be understood to denote the installation directory prefix where the SBW system was installed.
