This directory is used to archive biomolecular models
that have been found useful to the Bolouri Group,
and that are believed to be of sufficient utility
and validity to make available to the general public.
Any model definition file placed in this directory will
be publically visible on our external web server.  After
placing a file in this directory, it will be necessary
to edit the &quot;<code>index.xml</code>&quot; file
in this directory, in order to add a hyperlink to the
file from the list on the &quot;<code>index.xml</code>&quot;
web page.
