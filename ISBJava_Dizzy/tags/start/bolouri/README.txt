This directory is the main directory for Java classes 
developed by the Bolouri Group at the Institute for Systems Biology,
and for applications based on these classes.

For more information on the class libraries, please consult the file
  java/overview.html

For more information on the applications, please consult the file
  apps/README.txt

The "config" subdirectory contains global project XML entity
definitions for things like the intranet URL and staging directory.

-Stephen Ramsey
